# 🗺️ Roadmap Globale — LUCID

---

## Vision Temporelle

> **Mission :** Devenir la plateforme de référence pour l'expérience scolaire numérique en France, puis à l'international.

---

## 📍 Phase 1 — 3 Mois (Avril – Juin 2026) : "Consolider la Base"

**Thème :** Stabilisation technique + croissance organique au Lycée Henri Matisse

### Produit & Technique
- [ ] Finaliser le fine-tuning des modèles edge **Gemma E2B/E4B** avec les 10 000 exemples Gold Data
- [ ] Implémenter le **RAG on-device** pour l'injection de contexte scolaire
- [ ] Compléter la génération du dataset via Lucid Labs (objectif : toutes les tâches à 100%)
- [ ] Intégrer le modèle fine-tuné dans l'application mobile (inférence locale)
- [ ] Corriger les derniers bugs de synchronisation XP / gamification
- [ ] Ajouter le support Android (Google Play Store)
- [ ] Migrer progressivement de l'API Gemini vers le modèle edge

### Acquisition & Croissance
- [ ] Finaliser le concours Beta Test au Lycée Henri Matisse
- [ ] Déployer des QR codes dans d'autres lycées de l'académie de Nice
- [ ] Atteindre 500+ utilisateurs actifs
- [ ] Créer 10+ posts Instagram avec démonstration des fonctionnalités IA

### Métriques Cibles
| KPI | Objectif |
|---|---|
| Utilisateurs actifs | 500+ |
| DAU/MAU | >30% |
| Retention J7 | >40% |
| Dataset Gold | 10 000 exemples ✅ |

## 📍 Phase 2 — 6 Mois (Avril – Octobre 2026) : "Expansion Locale"

**Thème :** Conquête de l'académie de Nice + plateforme professeur opérationnelle

### Produit & Technique
- [ ] Lancer **Lucid Prof** en beta auprès de 10 professeurs pilotes
- [ ] Ajouter le support **École Directe** (2ème ENT de France)
- [ ] **IA edge 100% opérationnelle** : plus aucune dépendance à l'API Gemini
- [ ] Implémenter le mode **multi-joueur** (défis entre amis en temps réel)
- [ ] Ajouter la fonctionnalité **Carte Mentale** générée par IA
- [ ] Mise en place de notifications push intelligentes

### Acquisition & Croissance
- [ ] Déploiement dans 5+ lycées de l'académie de Nice
- [ ] Partenariat avec des associations de parents d'élèves
- [ ] Première campagne publicitaire Instagram/TikTok ciblée
- [ ] Présence à un événement éducatif régional

### Business
- [ ] Lancement du modèle **Freemium** : fonctionnalités de base gratuites, quiz IA illimités en Pro
- [ ] Premier pricing : Free / Pro (4.99€/mois ou 39.99€/an)
- [ ] Objectif : premiers revenus récurrents (MRR)

### Métriques Cibles
| KPI | Objectif |
|---|---|
| Utilisateurs actifs | 3 000+ |
| Établissements couverts | 5+ |
| Professeurs actifs (Lucid Prof) | 10+ |
| MRR | 500€+ |

## 📍 Phase 3 — 1 An (Avril 2026 – Avril 2027) : "Validation Nationale"

**Thème :** Passage à l'échelle nationale + structuration de l'entreprise

### Produit & Technique
- [ ] IA edge propriétaire (Gemma fine-tuné) en production complète, zéro API cloud
- [ ] RAG avancé : mémoire long-terme de l'élève (historique de sessions)
- [ ] Import de notes manuscrites scannées (OCR → IA pédagogique)
- [ ] API publique pour les établissements (intégration ENT)
- [ ] Dashboard établissement (métriques d'engagement par classe)
- [ ] Support multilingue (français → anglais, espagnol)

### Acquisition & Croissance
- [ ] Déploiement dans 3+ académies
- [ ] 20 000+ utilisateurs actifs
- [ ] Partenariats avec des éditeurs scolaires (Hachette, Nathan, Bordas)
- [ ] Présence au salon Educatec-Educatice

### Business & Structuration
- [ ] Création de la structure juridique (SAS)
- [ ] **Initier les discussions avec Index Éducation (Pronote)** pour une acquisition ou un partenariat stratégique
- [ ] Alternativement : lever de fonds Seed (200K-500K€) si l'acquisition n'aboutit pas
- [ ] Lancer l'offre **Enterprise** pour les établissements (licence annuelle par élève)
- [ ] Recruter 2-3 personnes clés (designer/DA, dev mobile, commercial B2B)

### Métriques Cibles
| KPI | Objectif |
|---|---|
| Utilisateurs actifs | 20 000+ |
| Établissements couverts | 50+ |
| MRR | 10 000€+ |
| Churn mensuel | <5% |

## 📍 Phase 4 — 5 Ans (2026 – 2031) : "Leader EdTech France"

**Thème :** Position dominante + expansion internationale

### Vision Produit
- [ ] **Lucid Marketplace** : Les professeurs partagent et vendent leurs modules pédagogiques
- [ ] **IA Tuteur Personnalisé** : Un modèle qui connaît l'historique complet de l'élève sur 3 ans pour un coaching ultra-ciblé
- [ ] **Lucid Live** : Dashboard en temps réel montrant l'activité du réseau mondial (devoirs appris, quiz résolus)
- [ ] **Simulations 3D interactives** pour les matières scientifiques
- [ ] **Partenariats avec les Ministères de l'Éducation** de pays francophones (Belgique, Suisse, Afrique francophone)

### Vision Business
- [ ] 500 000+ utilisateurs actifs
- [ ] Présence dans 5+ pays francophones
- [ ] ARR (Annual Recurring Revenue) de 2M€+
- [ ] Équipe de 20+ personnes
- [ ] Introduction de modèles d'abonnement famille et institutionnel

### Vision Impact
- [ ] Réduction mesurable du décrochage scolaire dans les établissements partenaires
- [ ] Publication d'études d'impact pédagogique
- [ ] Label "EdTech de confiance" décerné par le Ministère de l'Éducation

## ⚠️ Risques et Dépendances

| Risque | Impact | Mitigation |
|---|---|---|
| Pronote bloque le proxy API | 🔴 Critique | Proxy open-source (confiance), diversifier vers École Directe, négociation partenariat |
| Modèle edge de qualité insuffisante | 🟡 Élevé | Dataset validé par LLM-as-a-Judge (9/10), fallback sur Gemini API |
| **Les 2 fondateurs entrent en études sup** | 🔴 **Critique** | **Accélérer les discussions d'acquisition**, trouver un directeur opérationnel ou prendre une année sabbatique |
| Adoption lente des professeurs | 🟡 Moyen | Focus sur les élèves (B2C) d'abord, les profs suivront naturellement |
| Conformité RGPD challengée | 🟡 Moyen | Politique de confidentialité solide, proxy transparent, conseil juridique |
| Nom "LUCID" déjà pris / problème de marque | 🟢 Moyen | Vérification INPI + rebranding si nécessaire (pas une priorité immédiate) |
| Trésorerie insuffisante | 🟢 Faible (actuellement) | Coûts infrastructure très bas (Gemini gratuit), pas de salaires |
