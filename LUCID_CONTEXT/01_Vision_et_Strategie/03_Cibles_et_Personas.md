# 👥 Cibles et Personas — LUCID

---

## 1. Segmentation du Marché

LUCID opère sur un marché B2C (initialement) avec une vision B2B2C (à terme) :

| Segment | Rôle | Priorité |
|---|---|---|
| Élèves (End-Users) | Utilisateurs principaux de l'app mobile | 🔴 P0 — Critique |
| Professeurs (Créateurs) | Utilisateurs de Lucid Prof (back-office pédagogique) | 🟡 P1 — Stratégique |
| Parents (Observateurs) | Suivi des résultats de l'enfant (futur) | 🟢 P2 — Futur |
| Établissements scolaires (Acheteurs B2B) | Décideurs pour déploiements institutionnels | 🟢 P2 — Futur |

## 2. Personas — Utilisateurs (End-Users)

### 2.1. 🎯 Persona Principal : "Lucas" — L'Élève Connecté

| Attribut | Détail |
|---|---|
| **Âge** | 15-18 ans |
| **Niveau** | Lycée (Seconde → Terminale) |
| **Appareil** | iPhone (cœur de cible iOS) |
| **Profil Tech** | Digital native, utilise TikTok, Instagram, Discord quotidiennement |
| **Rapport à l'école** | Moyennement motivé, "fait ce qu'il faut" mais sans passion |
| **Douleurs** | Pronote est moche et ennuyeux, il ne sait pas comment réviser efficacement, les cours particuliers sont trop chers |
| **Motivations** | Compétition amicale, progression visible, gadgets/apps premium |
| **Objectif** | Améliorer ses notes sans que ça soit une corvée |

**🗣️ Citation type :** *"Si Pronote ressemblait à une vraie app, peut-être que j'y irais plus souvent."*

**Parcours type :**
1. Découvre LUCID via un ami, un QR code au lycée ou Instagram
2. Télécharge sur l'App Store, se connecte avec ses identifiants Pronote
3. Découvre ses notes dans une interface moderne → effet "wow"
4. Commence à gagner de l'XP, se compare avec ses amis
5. Utilise les quiz IA avant un contrôle → améliore ses notes
6. Devient utilisateur quotidien (rétention via la gamification)

---

### 2.2. 📚 Persona Secondaire : "Emma" — L'Élève Studieuse

| Attribut | Détail |
|---|---|
| **Âge** | 14-17 ans |
| **Niveau** | Collège/Lycée (3ème → Première) |
| **Profil** | Bonne élève, proactive, cherche toujours à s'améliorer |
| **Douleurs** | Les plateformes de soutien scolaire (Kartable) sont trop génériques et pas liées à SES cours |
| **Motivations** | Réussir, comprendre en profondeur, avoir les meilleurs outils |
| **Objectif** | Avoir un outil IA qui s'adapte exactement à son programme |

**🗣️ Citation type :** *"Je veux un outil qui utilise mes vrais cours pour me faire réviser, pas des exercices random."*

**Valeur LUCID pour elle :** Les quiz et fiches de révision sont générés à partir de son propre cahier de texte Pronote. C'est hyper-personnalisé.

---

### 2.3. 🎮 Persona Tertiaire : "Nathan" — Le Compétiteur

| Attribut | Détail |
|---|---|
| **Âge** | 15-17 ans |
| **Profil** | Joueur, aime les classements, streaks et achievements |
| **Douleurs** | L'école est ennuyeuse, il ne voit pas la progression |
| **Motivations** | Être premier au classement, avoir tous les badges |
| **Objectif** | Transformer l'école en jeu |

**🗣️ Citation type :** *"Si je gagne de l'XP en bossant, ça change tout."*

**Valeur LUCID pour lui :** Le leaderboard, les badges, les niveaux et le système de compétition entre amis.

## 3. Personas — Professeurs (Créateurs de Contenu)

### 3.1. 👨‍🏫 "M. Dupont" — Le Prof Moderne

| Attribut | Détail |
|---|---|
| **Âge** | 30-50 ans |
| **Matière** | Toutes (priorité : Maths, Français, Hist-Géo, SVT, Physique-Chimie) |
| **Profil Tech** | À l'aise avec le numérique, utilise déjà des outils comme Kahoot, Quizlet |
| **Douleurs** | Créer des quiz/exercices prend des heures. Pronote ne permet pas de créer du contenu interactif |
| **Motivations** | Innover dans sa pédagogie, engager ses élèves davantage |
| **Objectif** | Un outil qui génère des exercices de qualité à partir de ses propres cours |

**Valeur LUCID pour lui :** Via **Lucid Prof**, il peut numériser son cours (manuellement ou via IA), et les contenus générés sont automatiquement disponibles pour les élèves dans l'app.

## 4. Personas — Acheteurs / Decision Makers

### 4.1. 🏫 "Mme Martin" — La Proviseure

| Attribut | Détail |
|---|---|
| **Rôle** | Chef d'établissement, décideur pour les outils numériques |
| **Douleurs** | Les élèves sont désengagés, le taux d'échec augmente, les outils institutionnels sont archaïques |
| **Motivations** | Moderniser l'établissement, montrer de l'innovation pédagogique |
| **Freins** | RGPD, sécurité des données, validation académique, budget |
| **Ce qui la convainc** | Compliance RGPD documentée, transparence (proxy open-source), résultats mesurables, dépôt de brevet |

**🗣️ Citation type :** *"Si l'outil est conforme RGPD et que les élèves l'adoptent vraiment, on le déploie."*

### 4.2. 👨‍👩‍👦 "Les Parents"

| Attribut | Détail |
|---|---|
| **Rôle** | Financeurs potentiels (offre premium), observateurs |
| **Douleurs** | Ne savent pas comment aider leur enfant, n'ont pas les compétences pour certaines matières |
| **Motivations** | Voir leur enfant réussir, avoir de la visibilité sur sa progression |
| **Ce qui les convainc** | L'IA ne donne pas les réponses (pédagogie active), transparence sur les données, prix abordable |

## 5. Matrice Cible vs Valeur

| Persona | Fonctionnalité LUCID la plus forte | Canal d'acquisition |
|---|---|---|
| L'Élève Connecté | Hub scolaire moderne + Gamification | QR codes au lycée, Instagram, bouche-à-oreille |
| L'Élève Studieuse | Quiz IA + Fiches de révision contextuelles | App Store organic, SEO |
| Le Compétiteur | Leaderboard + Badges + XP | Concours (Beta Test), viral entre amis |
| Le Prof Moderne | Lucid Prof (studio de création assisté par IA) | Démarchage B2B, formation continue |
| La Proviseure | Tableau de bord (Lucid Dashboard) + RGPD | Événements éducatifs, partenariats académiques |
| Les Parents | Suivi parental (futur) | SEO, App Store, recommandation child |

## 6. Marché Adressable

| Métrique | Valeur |
|---|---|
| **TAM** (Total Addressable Market) | ~12M d'élèves en France (primaire → terminale) |
| **SAM** (Serviceable Addressable Market) | ~5.5M d'élèves collège/lycée utilisant Pronote |
| **SOM** (Serviceable Obtainable Market — Y1) | ~10 000 élèves (1 académie, bouche-à-oreille + concours) |

> 💡 **Note :** Pronote est utilisé par >80% des établissements secondaires en France. C'est un marché captif massif.
