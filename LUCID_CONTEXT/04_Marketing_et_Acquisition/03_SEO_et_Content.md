## 2. Piliers de Contenu

### Pilier 1 : "IA et Éducation" 🧠
- Comment l'IA peut aider à réviser sans tricher
- Différence entre ChatGPT et une IA pédagogique
- Le questionnement socratique : pourquoi c'est efficace
- Fine-tuning de modèles IA pour l'éducation

### Pilier 2 : "Méthodes de Révision" 📚
- Les flashcards : la science de la mémorisation active
- Techniques de révision efficaces pour le bac
- Comment organiser ses révisions sans stress
- Quiz ou relecture : qu'est-ce qui marche vraiment ?

### Pilier 3 : "Gamification Scolaire" 🎮
- Comment la gamification améliore les résultats scolaires
- XP, badges et niveaux : le jeu au service de l'apprentissage
- Études de cas : Duolingo et l'engagement utilisateur
- Le leaderboard scolaire : compétition saine ou stress ?

### Pilier 4 : "Notre Histoire" 📖
- De l'idée au brevet : l'histoire de LUCID
- Lauréat Science Factor : notre parcours
- Par des élèves, pour des élèves : pourquoi ça change tout
- Le quotidien d'une startup EdTech lycéenne

## 4. SEO Technique (Site Officiel)

### 4.1. Optimisations Actuelles

| Élément | Statut | Détail |
|---|---|---|
| Title Tags | ✅ | Titres descriptifs par page |
| Meta Descriptions | 🟡 À améliorer | Certaines pages manquent de descriptions |
| Structure H1/H2 | ✅ | Hiérarchie respectée |
| HTML Sémantique | ✅ | `<section>`, `<main>`, `<footer>` |
| Responsive | ✅ | Mobile-first |
| HTTPS | ✅ | Via Vercel |
| Performance | ✅ | Vite build optimisé |

### 4.2. Améliorations Prévues

| Action | Priorité |

| Ajouter le sitemap.xml | 🔴 P0 |

| Structured data (JSON-LD) pour l'app | 🟡 P1 |
| Créer la page /blog avec contenu SEO | 🟡 P1 |
