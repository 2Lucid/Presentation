# 📖 Glossaire — LUCID

---

## A

| Terme | Définition |
|---|---|
| **AXEPI** | Cabinet de propriété intellectuelle partenaire de LUCID pour le dépôt de brevet |
| **Activation** | Moment où un utilisateur atteint son premier "aha moment" (ex: premier quiz IA) |
| **ARR** | Annual Recurring Revenue — Revenus récurrents annuels |
| **ASO** | App Store Optimization — Optimisation de la fiche App Store |
| **AsyncStorage** | Système de stockage persistant local pour React Native (clé/valeur) |
| **Atrium** | ENT (Espace Numérique de Travail) de l'académie de Nice |
| **Autopilot** | Mode de génération automatique continue dans Lucid Labs |

## B

| Terme | Définition |
|---|---|
| **Badge** | Récompense visuelle débloquable dans le système de gamification |
| **B2C** | Business to Consumer — Vente directe aux élèves (modèle actuel) |
| **B2B2C** | Business to Business to Consumer — Vente via les établissements aux élèves (modèle futur) |

## C

| Terme | Définition |
|---|---|
| **CAC** | Customer Acquisition Cost — Coût d'acquisition d'un client |
| **ChatML** | Format de dataset pour le fine-tuning de LLM (`<\|im_start\|>system...`) |
| **Churn** | Taux de désabonnement / perte d'utilisateurs |
| **CLG** | Community-Led Growth — Croissance tirée par la communauté |
| **CPI** | Cost per Install — Coût par installation |
| **Combo** | Combinaison unique (task_type + subject + level + mode + topic) dans le pipeline Labs |

## D

| Terme | Définition |
|---|---|
| **DAU** | Daily Active Users — Utilisateurs actifs quotidiens |
| **Dataset** | Ensemble de données structurées pour entraîner un modèle IA |
| **D7 Retention** | % d'utilisateurs actifs 7 jours après l'installation |
| **Deterministic UUID** | ID généré de manière reproductible à partir d'inputs constants (nom+classe+école) |

## E

| Terme | Définition |
|---|---|
| **EAS** | Expo Application Services — Services de build et déploiement pour Expo |
| **Edge AI** | Architecture où l'inférence IA se fait directement sur l'appareil de l'utilisateur (on-device), sans dépendance au cloud |
| **Edge Function** | Fonction serverless exécutée au plus proche de l'utilisateur (Supabase/Deno) |
| **ENT** | Espace Numérique de Travail — Portail web institutionnel des établissements |
| **Enterprise** | Offre B2B pour les établissements scolaires |

## F

| Terme | Définition |
|---|---|
| **Flashcard** | Carte de mémorisation recto/verso |
| **Freemium** | Modèle économique gratuit avec options payantes |
| **Fine-tuning** | Entraînement d'un modèle IA pré-entraîné sur des données spécifiques |

## G

| Terme | Définition |
|---|---|
| **Gamification** | Utilisation de mécaniques de jeu dans un contexte non-ludique |
| **Gemini** | Modèle LLM cloud de Google, utilisé temporairement en phase beta |
| **Gemma** | Modèle LLM edge de Google (E2B / E4B), fine-tuné et embarqué sur l'appareil de l'élève. Cœur de la stratégie IA LUCID |
| **Glassmorphism** | Style de design avec fond translucide, flou et bordures subtiles |
| **Gold Data** | Exemple de dataset ayant reçu un score ≥ 9/10 par le LLM-as-a-Judge |
| **GTM** | Go-To-Market — Stratégie de mise sur le marché |

## J

| Terme | Définition |
|---|---|
| **JSONL** | JSON Lines — Format de fichier avec un objet JSON par ligne (pour le fine-tuning) |
| **Judge** | LLM-as-a-Judge — Système de validation automatique des exemples par un LLM |

## K

| Terme | Définition |
|---|---|
| **Key Rotation** | Système de rotation automatique des clés API Gemini pour gérer les quotas |
| **KPI** | Key Performance Indicator — Indicateur clé de performance |

## L

| Terme | Définition |
|---|---|
| **Leaderboard** | Classement des utilisateurs par XP |
| **LLM** | Large Language Model — Grand modèle de langage (ex: Gemini, GPT) |
| **LTV** | Lifetime Value — Valeur totale d'un utilisateur sur sa durée de vie |
| **Lucid Labs** | Plateforme R&D interne pour la génération de datasets IA |
| **Lucid Prof** | Plateforme web pour les professeurs (création de contenu) |
| **Lucid Dashboard** | Panneau d'administration interne |
| **Lucid Redirect** | Page HTML de redirection automatique vers l'App Store/Play Store |

## M

| Terme | Définition |
|---|---|
| **MAU** | Monthly Active Users — Utilisateurs actifs mensuels |
| **Max XP Wins** | Stratégie de résolution de conflits : la valeur XP la plus haute est conservée |
| **Monorepo** | Repository unique contenant plusieurs projets |
| **MRR** | Monthly Recurring Revenue — Revenus récurrents mensuels |

## P

| Terme | Définition |
|---|---|
| **Pawnote** | Bibliothèque Node.js pour communiquer avec le protocole Pronote |
| **Pipeline** | Séquence de traitement automatisée (Contexte → Génération → Jugement) |
| **Pronote** | Logiciel de gestion de vie scolaire (Index Éducation), utilisé par >80% des collèges/lycées |
| **Proxy** | Serveur intermédiaire qui relaie les requêtes vers Pronote |

## Q

| Terme | Définition |
|---|---|
| **QCM** | Question à Choix Multiple |

## R

| Terme | Définition |
|---|---|
| **RAG** | Retrieval-Augmented Generation — Technique qui enrichit les prompts IA avec du contexte récupéré (cours Pronote de l'élève). Implémenté on-device dans LUCID |
| **Recall (mode)** | Mode quiz qui teste les matières du lendemain |
| **RLS** | Row Level Security — Sécurité au niveau de la ligne (PostgreSQL/Supabase) |
| **RPC** | Remote Procedure Call — Appel de fonction à distance (PostgreSQL function) |

## S

| Terme | Définition |
|---|---|
| **SAS** | Société par Actions Simplifiée — Structure juridique prévue |
| **Science Factor** | Concours national d'innovation (LUCID = Lauréat Prix Lycée + Prix Orange) |
| **Server Action** | Fonction côté serveur dans Next.js (`"use server"`) |
| **Socratique** | Méthode pédagogique basée sur le questionnement guidé |
| **Supabase** | Backend-as-a-Service (PostgreSQL + Auth + Storage + Edge Functions) |
| **syncFromSupabase** | Méthode qui synchronise le store local avec les données Supabase |

## T

| Terme | Définition |
|---|---|
| **TAM** | Total Addressable Market — Marché total adressable |
| **Tribunal** | Nom informel du système LLM-as-a-Judge dans Lucid Labs |
| **Tuteur Socratique** | Mode IA conversationnel qui guide par le questionnement |

## U

| Terme | Définition |
|---|---|
| **Unfair Advantage** | Avantage concurrentiel déloyal (difficile à copier) |

## X

| Terme | Définition |
|---|---|
| **XP** | Experience Points — Points d'expérience dans le système de gamification |
