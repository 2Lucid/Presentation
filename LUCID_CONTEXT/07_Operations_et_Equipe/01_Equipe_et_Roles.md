## 3. Exit Strategy & Avenir

> ⚠️ **Contexte critique :** Les 2 fondateurs entrent en études supérieures à la rentrée 2026-2027. Le temps disponible pour le développement va drastiquement diminuer.

### 3.1. Options sur la Table

| Option | Description | Pour | Contre |
|---|---|---|---|
| **🏆 Vente / Acquisition** | Vendre LUCID à un acteur de l'EdTech (ex: **Index Éducation** / Pronote, Docaposte, Hachette, Google for Education) tout en conservant une part minoritaire | Cash immédiat, pérennité du projet, ressources de l'acquéreur | Perte de contrôle, valorisation potentiellement basse si trop tôt |
| **💰 Levée de Fonds** | Trouver des investisseurs (Business Angels, fonds EdTech, BPI France) | Financer le développement, recruter, garder le contrôle majoritaire | Dilution, pression sur la croissance, complexité juridique |
| **⏸️ Pause Études** | Un ou les deux fondateurs prennent une année sabbatique pour se consacrer au projet | Focus total, potentiel d'accélération rapide | Risque sur le parcours académique |
| **🤝 Licence / Partenariat** | Licencier la technologie (IA edge + gamification) à un acteur qui la distribue | Revenus récurrents, pas besoin de gérer l'exploitation | Moins de contrôle sur le produit final |
| **👥 Recruter un Directeur** | Trouver un CEO/COO opérationnel qui reprend la gestion quotidienne | Les fondateurs gardent la vision, délèguent l'exécution | Difficile de trouver la bonne personne, coûteux |

### 3.2. Scénario Idéal

1. **Court terme (3-6 mois)** : Atteindre une traction suffisante (5 000+ utilisateurs, modèle fine-tuné opérationnel) pour démontrer de la valeur
2. **Pitch** : Approcher Index Éducation (Pronote) avec une proposition claire : "Nous avons résolu le problème que votre interface ne résout pas — la gamification + l'IA pédagogique — tout est prêt, le brevet est déposé"
3. **Deal** : Vente avec conservation d'une part minoritaire (~10-20%) et potentiellement un rôle de consultant technique
4. **Résultat** : Les fondateurs poursuivent leurs études avec un capital initial et la satisfaction de voir LUCID grandir avec les moyens d'un grand groupe

### 3.3. Cibles d'Acquisition Potentielles

| Cible | Pourquoi ils seraient intéressés | Synergies |
|---|---|---|
| **Index Éducation** (Pronote) | Moderniser leur interface, intégrer l'IA, la gamification — exactement ce qui leur manque | LUCID = le frontend moderne de Pronote |
| **Docaposte** (filiale La Poste) | Opérateur de plateformes numériques éducatives | Distribution massive |
| **Hachette / Nathan** | Éditeurs scolaires cherchant la digitalisation | Contenu pédagogique × technologie IA |
| **Bescherelle / Hatier** | Même logique | Marque forte + tech LUCID |
| **Un fond EdTech** (Brighteye, Owl Ventures) | Investir dans le potentiel, pas acquérir | Financement + mentorat |

### 3.4. Valorisation Estimée (Très Early-Stage)

| Facteur de Valorisation | Valeur |
|---|---|
| Brevet déposé (AXEPI) | Atout fort |

## 6. Culture d'Équipe

### Valeurs

| Valeur | Signification |
|---|---|
| **Speed > Perfection** | On ship vite, on itère. Un MVP imparfait bat un plan parfait jamais exécuté |
| **User-First** | L'élève est au centre de chaque décision |
| **Transparency** | Code open-source pour le proxy, politique de confidentialité claire |
| **Passion > Process** | On est là parce qu'on croit au projet, pas pour remplir des Jira tickets |
| **Edge-First** | L'IA doit tourner sur le téléphone de l'élève, pas dans le cloud |
